(1)全国新冠肺炎疫情总体状况统计并展示；

(4)全国新冠肺炎疫情新增人数趋势通过曲线图展示；
 
(5)非湖北地区城市确诊人数TOP5柱状图排行展示； 
(6)将从百度疫情热搜爬取到的条目通过提取关键词制作词云图展示。
